# MCP Tools

> **Владелец:** DoctorM&Ai | **Статус:** active

## Описание

Инструменты для Model Context Protocol (MCP) — серверы и утилиты для интеграции с LLM. Предоставляет готовые бинарные утилиты для подключения MCP-совместимых инструментов к агентам лаборатории.

## Быстрый старт

### Требования

- Linux/macOS
- OpenClaw (для интеграции с агентами)

### Установка

```bash
cd /root/LabDoctorM/projects/mcp-tools
# Утилиты готовы к использованию из bin/
ls bin/
```

### Запуск

```bash
# Конкретные команды зависят от используемого инструмента
# См. документацию конкретной утилиты в bin/
```

## Архитектура

- **Стек:** Bash, Python (зависит от утилиты)
- **Компоненты:**
  - `bin/` — исполняемые утилиты MCP-инструментов
